Responsive tables

What does this thing do?

  1. Checks when the table is larger then it's parent (parent created by jQuery)
  2. Checks if there's table headers with the data-toggle-order attribute and if so removes the th with corresponding td with the lowest data-toggle-order first
  3. If there are multiple th's with the same data-toggle-order value it removes the most right one first
  4. When removed it saves the current width as "leave" value so when there's space again we can put the td's back
  5. When there's a data-has-toggle attribute on one of the th's it puts a toggle button to that td
  6. When no data-has-toggle is found it adds the toggle to the first td
  7. When the toggle area is open and screen is being resized smaller it adds the td that is being hidden to the toggle area. In the right order
  8. When toggle area is open and we are making our screen bigger it removes the values one at a time
  9. When there are no more things to show in the toggle area it removes the area and the arrow
  10. Works with multiple tables at a page
